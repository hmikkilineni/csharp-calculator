# C# Calculator

A console app built in C# that performs basic arithmetic operations.

## Features
- Addition, Subtraction, Multiplication, Division
- User input via console
- Error handling for invalid input
- Division by zero protection

## Error Handling
- try/catch/finally blocks
- FormatException — catches letters instead of numbers
- DivideByZeroException — catches division by zero
- finally — always prints goodbye message

## What I Learned
- Variables and data types
- String interpolation
- Conditionals (if/else)
- User input with Console.ReadLine()
- try/catch error handling

## Tech
- C# / .NET 8
